# 節目聖經欄位完整指南

## 📋 概述

節目聖經（Show Bible）是您播客節目的核心文檔，包含所有重要的節目設定和風格指引。這份指南詳細說明每個欄位的用途和填寫建議。

---

## 🎯 基本資訊欄位

### 1. 節目名稱 *（必填）*
- **用途**：節目的正式名稱
- **範例**：`每日靈修時光`、`商業智慧對談`
- **提示**：簡潔有力，易於記憶

### 2. 節目描述 *（必填）*
- **用途**：詳細說明節目的主要內容和目標
- **範例**：
  ```
  這是一個專為忙碌都市人設計的靈修節目，
  每天15分鐘，透過聖經經文、生活應用和反思問題，
  幫助聽眾在繁忙生活中找到心靈的平靜與力量。
  ```
- **提示**：清楚說明節目價值和獨特之處

---

## 🎨 風格與定位

### 3. 節目格式
- **用途**：說明節目的呈現形式
- **範例**：
  - `獨白式講述`
  - `一對一訪談`
  - `多人圓桌討論`
  - `混合式（講述 + 訪談）`

### 4. 語調風格
- **用途**：定義節目的說話方式和氛圍
- **範例**：
  - `輕鬆幽默、親切自然`
  - `專業嚴謹、知識性強`
  - `溫暖鼓勵、充滿同理心`
  - `激勵人心、富有熱情`

### 5. 目標受眾
- **用途**：明確節目的主要聽眾群
- **範例**：
  - `25-40歲職場女性`
  - `對信仰有興趣的年輕人`
  - `創業者和商業人士`
  - `親子教育關注者`

---

## ⏱️ 結構與時長

### 6. 節目時長
- **用途**：每集的預計長度
- **範例**：
  - `15-20分鐘（通勤友善）`
  - `30-45分鐘（標準長度）`
  - `60-90分鐘（深度對談）`

### 7. 節目架構
- **用途**：詳細的時間分配和環節安排
- **範例**：
  ```
  00:00-02:00  開場音樂 + 問候
  02:00-05:00  本集主題介紹
  05:00-20:00  核心內容講解
  20:00-25:00  實踐應用建議
  25:00-28:00  反思問題 + 禱告
  28:00-30:00  結尾 + 下集預告
  ```

---

## 👥 人物與角色

### 8. 主持人資訊
- **用途**：主持人的背景和特色
- **範例**：
  ```
  王小明，資深靈修導師，擁有15年教會服事經驗。
  說話溫和、善於聆聽，擅長用生活化的例子解釋聖經真理。
  ```

### 9. 角色/人物設定 ⭐ **新增**
- **用途**：詳細描述節目中的所有角色設定
- **包含**：
  - 主持人個性與背景
  - 來賓類型與條件
  - 角色互動方式
- **範例**：
  ```
  【主持人】
  • 名字：張大衛
  • 個性：開朗健談、好奇心強、善於提問
  • 背景：10年媒體經驗，深度訪談專家
  • 風格：以真誠的好奇心引導對話，不刻意營造權威感

  【來賓類型】
  • 業界專家（70%）
  • 實踐者/創業者（20%）
  • 跨界人士（10%）

  【互動方式】
  • 輕鬆自然的對話，避免過於正式
  • 適時追問，挖掘深層洞見
  • 鼓勵分享個人故事和失敗經驗
  ```

## 🎬 特色與風格

### 10. 參考節目 ⭐ **新增**
- **用途**：列出欣賞的節目及其可借鑒的元素
- **包含**：
  - 參考的節目名稱
  - 喜歡的風格元素
- **範例**：
  ```
  【國際參考】
  • The Tim Ferriss Show
    - 深度訪談、挖掘細節
    - 快問快答環節

  • How I Built This
    - 故事性強、情感共鳴
    - 關注創業歷程的真實面

  【華語參考】
  • 《得到》- 知識濃縮、節奏明快
  • 《冬吳相對論》- 思想深度、啟發性

  【借鑒元素】
  ✓ 開場用故事或問題吸引注意
  ✓ 穿插個人經歷增加真實感
  ✓ 結尾提供可行動的建議
  ```

### 11. 特色環節 ⭐ **新增**
- **用途**：節目中的固定單元和特色設計
- **包含**：
  - 固定單元
  - 互動環節
- **範例**：
  ```
  【固定單元】
  • "本週金句"：每集分享一句啟發性經文
  • "行動時刻"：提供 3 個具體的實踐步驟
  • "聽眾來信"：每兩週回覆一封聽眾問題

  【互動環節】
  • 開場提問：用一個引人思考的問題開始
  • 中場暫停：鼓勵聽眾暫停思考自己的答案
  • 結尾挑戰：給聽眾一個本週的行動挑戰

  【特色元素】
  • 每集都有一個"微故事"（2-3分鐘）
  • 使用"對話式"語言，如同與朋友聊天
  • 不定期的"幕後花絮"分享
  ```

### 12. 音樂/音效 ⭐ **新增**
- **用途**：定義節目的聲音氛圍
- **包含**：
  - 開場音樂風格
  - 背景音樂類型
- **範例**：
  ```
  【開場音樂】
  • 風格：輕快的吉他旋律 + 溫暖的弦樂
  • 時長：15秒
  • 情緒：充滿希望、迎接新一天的感覺

  【背景音樂】
  • 主要內容：柔和的鋼琴或環境音樂（音量 20%）
  • 故事環節：略帶情感的弦樂（音量 15%）
  • 反思時刻：寧靜的大自然音效（流水、鳥鳴）

  【轉場音效】
  • 環節切換：輕柔的鐘聲
  • 重點強調：簡短的「叮」聲

  【結尾音樂】
  • 與開場音樂呼應，但更溫和
  • 漸弱淡出，留給聽眾思考空間
  ```

### 13. 品牌定位 ⭐ **新增**
- **用途**：確立節目的核心價值和品牌形象
- **包含**：
  - 節目口號
  - 核心價值
- **範例**：
  ```
  【節目口號】
  "每日一刻，心靈充電"

  【核心價值】
  • 真實 - 不迴避困難，分享真實掙扎
  • 溫暖 - 用同理心陪伴聽眾
  • 啟發 - 不只講道理，更重視啟發思考
  • 實用 - 每集都有可實踐的行動建議

  【品牌個性】
  • 像一位溫暖的朋友，而非高高在上的導師
  • 既專業又平易近人
  • 關注實際生活，不空談理論

  【獨特價值主張】
  "在忙碌的生活中，給你一個暫停鍵。
   15分鐘的心靈對話，讓信仰成為生活的力量。"

  【不要做的事】
  ✗ 說教式的單向輸出
  ✗ 過度使用宗教術語
  ✗ 迴避實際問題，只談理想
  ```

## 💡 使用建議

### 如何填寫節目聖經

1. **從核心開始**
   - 先確定節目的核心價值和目標受眾
   - 想清楚：「為什麼聽眾要聽這個節目？」

2. **保持一致性**
   - 所有欄位應該互相呼應
   - 風格、定位、內容應該統一

3. **具體而非抽象**
   - 用具體的例子和描述
   - 避免模糊的形容詞

4. **可以持續更新**
   - 節目聖經不是一成不變的
   - 隨著節目發展，可以調整優化

### AI 腳本生成

當您完整填寫節目聖經後，AI 會根據這些資訊生成更符合您節目風格的腳本。欄位填寫得越詳細，生成的腳本就越貼合您的需求。

## 🎯 快速開始範例

### 完整範例：《每日靈修時光》

```markdown
【基本資訊】
節目名稱：每日靈修時光
節目描述：為忙碌都市人設計的15分鐘靈修節目，透過聖經經文、生活應用和反思問題，幫助聽眾在繁忙生活中找到心靈的平靜與力量。

【風格定位】
節目格式：獨白式講述 + 引導式反思
語調風格：溫暖親切、充滿同理心
目標受眾：25-45歲的職場基督徒
節目時長：12-15分鐘

【結構安排】
00:00-01:00  溫暖問候 + 簡短禱告
01:00-03:00  當日經文朗讀與背景
03:00-08:00  經文解析與生活應用
08:00-11:00  反思問題引導
11:00-13:00  總結禱告
13:00-15:00  祝福與下集預告

【人物角色】
主持人：李心怡，教會小組長，善於將信仰與生活連結

角色設定：
• 主持人：溫柔但堅定，像心靈導師般的朋友
• 說話風格：緩慢清晰，給聽眾思考空間
• 個性特質：真誠、同理心強、不說教

【特色風格】
參考節目：
• 《每日聽見神》- 簡短有力的靈修分享
• 《Daily Audio Bible》- 溫暖的陪伴感

特色環節：
• "今日金句"：精選一句經文
• "暫停思考"：三個反思問題
• "行動挑戰"：一個具體的實踐建議

音樂音效：
• 開場：柔和的鋼琴旋律
• 背景：自然音效（流水、鳥鳴）
• 結尾：安詳的弦樂

品牌定位：
• 口號："每日一刻，與神相遇"
• 核心價值：真實、溫暖、實用
• 定位：心靈陪伴者，而非權威教導者
```
